# 机器学习（常用科学计算库）定位&目标

## 定位

作为人工智能领域(数据挖掘/机器学习方向)的基础课程，为长期从事人工智能方向打下坚实的基础。

## 目标

- 掌握机器学习基础环境的安装和使用
- 利用常用的科学计算库对数据进行分析
- 利用常用的科学计算库对数据进行可视化展示
