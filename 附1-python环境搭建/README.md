# 前言

环境配置和环境搭建看似基础，但对于绝大多数人，在学习初期还是没办法快速且正确的配置环境，导致学习进度滞后，或者干脆放弃需学习python。

为了让广大pyhton初学者能够把更多的精力和努力花费在知识学习上，我们决定整理一套集各大主流平台于一身的定制化pyhton环境解决方案，在解决学习者烦恼的基础上尽最大努力提升学习者的知识体验。
